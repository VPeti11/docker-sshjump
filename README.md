# docker-sshjump
Docker container that can jump ssh 

# How to run

### Clone the repo
git clone https://github.com/vpeti11/docker-sshjump.git

### Add your ssh private key
Use file id_rsa in the same folder as the Dockerfile

### Modify ip
In Dockerfile

### Run

  docker compose up --build -d

### Done
Listening on port 5774
